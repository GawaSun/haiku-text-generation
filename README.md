# Haiku Generation NLP Project

Built and evaluated language models (Bigram baseline and GPT-style transformers) to generate haiku poetry with a 5-7-5 syllable structure. Implemented dataset preprocessing, tokenization experiments, and model training in Python using PyTorch. Compared models using quantitative metrics and human evaluation, identifying a word-level GPT architecture as the most effective for coherent text generation.

## Tech Stack
- Python
- PyTorch
- Jupyter Notebook

## Files
- `Final_Project_Haiku(3).ipynb` – notebook with preprocessing, modeling, and evaluation
- `Haiku Generation Report.pdf` – final project report
